SmallCage

a simple website generator

Quick Start

Install

RubyGems を使ってインストールします。


$ gem install smallcage

Create Project

smc importコマンドでプロジェクトのテンプレートをインポートします。


$ mkdir mysite
$ cd mysite
$ smc import default,standard

defaultは最低限のファイルを含むプロジェクトテンプレートです。standardは一般的なサイトのプロジェクトテンプレートです。

Update

smc updateコマンドで、サイトを構築します。


$ smc update

(filename).smcファイルから、(filename)が生成されます。

Start HTTP Server


$ smc server . 8080

http://localhost:8080/ にアクセスしてサイトが表示されることを確認します。サーバを停止する時は、Control-Cを押します。

Start Auto Update Daemon


$ smc auto

*.smc ファイルを修正した時に自動で smc update を実行します。
